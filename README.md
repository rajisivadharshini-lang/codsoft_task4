Movie Recommendation System

Project Description

This project is a simple Movie Recommendation System developed using Python. The system recommends movies based on the genre entered by the user.

Features

- Recommends Action movies
- Recommends Comedy movies
- Recommends Drama movies
- User-friendly interface

Technologies Used

- Python

How to Run

1. Open the Python file.
2. Run the program.
3. Enter a genre such as:
   - action
   - comedy
   - drama
4. The program displays recommended movies.

Example Output

Enter genre: action

Recommended Movies:

- Leo
- Vikram
- Kaithi
